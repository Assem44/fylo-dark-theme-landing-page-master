# Fylo Dark Theme Landing Page

A responsive, dark-themed landing page for the Fylo file storage service. It showcases product features, productivity benefits, testimonials, and a call-to-action for early access sign-up.

## Features

* **Modern Layouts:** Heavy use of **CSS Grid** for the features section and testimonial cards, ensuring a rock-solid structure.
* **Advanced Styling:** Utilizing **CSS Custom Properties (Variables)** for consistent theme management and easier maintenance.
* **Form Validation:** Custom-built email validation with immediate visual feedback for the user (Error/Success states).
* **Interactive UI:** Polished hover effects on all buttons, links, and social media icons to enhance User Experience (UX).

---

## Built With

- **Semantic HTML5:** For better accessibility and SEO structure.
- **CSS Grid & Flexbox:** Used strategically to handle complex alignments.
- **JavaScript:** To manage form submission and validation logic.
