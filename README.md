# Fylo Dark Theme Landing Page

## Features

* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views (Mobile-first approach).
* **Modern Layouts:** Utilizes **CSS Grid** for the complex features section and testimonial grid.
* **Interactive UI:** Custom hover states for buttons, links, and social media icons for better UX.
* **Email Validation:** Includes a functional CSS-based validation UI for the "Early Access" form.
* **Dark Theme:** Built with a consistent dark-mode palette using CSS custom properties (variables).
